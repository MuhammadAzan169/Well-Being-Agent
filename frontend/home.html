How It Works

A sophisticated 10-step pipeline processes every query to ensure safe, accurate, and empathetic responses.

1

Language Detection

Your input is analyzed for Urdu script characters, Roman Urdu patterns, or English — automatically routing to the right language pipeline.

2

Safety Screening

Crisis detection checks for suicidal ideation or self-harm, providing immediate helpline resources. Off-topic queries are filtered out.

3

Smart Cache Lookup

Exact-match and fuzzy similarity search checks if a similar question was answered recently — delivering instant responses when possible.

4

Knowledge Retrieval

Semantic vector search retrieves the top-5 most relevant passages from the breast cancer knowledge base using sentence-transformers embeddings.

5

Empathetic Response Generation

Emotional analysis calibrates the tone, then a carefully engineered prompt guides the LLM to generate a warm, accurate, language-appropriate answer.
